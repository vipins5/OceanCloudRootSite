[Introduction] We’re updating how entities inserted from Add Work Content under the + menu behave in Microsoft 365 Copilot and Copilot Chat. With this change, entity names (such as document titles, people names, or other referenced objects) will no longer be inserted as text within the input box, making it easier for users to craft clean prompts without needing to manually delete entity labels. This feature is available to both users with a Microsoft 365 Copilot license (Microsoft 365 Copilot users) and users without a Microsoft 365 Copilot license (Copilot Chat users), but the experience varies.
